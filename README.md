# 💻 VS Code Portfolio - Developer Portfolio

A unique developer portfolio website designed to look and function exactly like Visual Studio Code interface.

## 📖 About The Project

This portfolio mimics the VS Code interface with working tabs, file explorer, interactive terminal, and keyboard shortcuts. Perfect for developers to showcase their skills in a creative, tech-inspired way.

## ✨ Features

- 🎨 VS Code inspired user interface
- 📁 File explorer with multiple file types
- 📄 Working tabs system (Home, About, Projects, Contact)
- ⌨️ Keyboard shortcuts (Ctrl+S, Ctrl+Shift+P, Ctrl+`)
- 🖥️ Interactive terminal with typing animation
- 📊 Status bar with live clock
- 🎯 Activity bar with different views
- 📱 Responsive design
- 🎨 Syntax highlighting for code blocks
- 🔍 Git integration simulation

## 🚀 How To Use

**1. Run the Application**

Open `index.html` in your web browser or use Live Server in VS Code.

**2. Navigate the Interface**

- Click files in explorer to open different tabs
- Click activity bar icons to switch views
- Use keyboard shortcuts for commands

**3. Keyboard Shortcuts**

| Shortcut | Action |
|----------|--------|
| Ctrl + S | Save simulation |
| Ctrl + Shift + P | Command palette |
| Ctrl + ` | Toggle terminal |
